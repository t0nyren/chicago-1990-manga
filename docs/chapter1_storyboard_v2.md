# 《芝加哥1990》第一话 · 分镜设计 v2

> v1 反馈：读者看完不懂"他们在做什么 / 为什么"。
> v2 起按 [narrative_sop_v2.md](./narrative_sop_v2.md) 7 件套 + 70% 规则全章重写。

> **每页强制要素**（重复读 SOP §5.1）：
> - ① 场景切换处 **时空 caption**
> - ② 关键物件 **特写镜头** + 提醒 caption（粉西装 / $20 / 麦克风 / 花头巾）
> - ③ 情绪节点 **反应镜头独立成格**
> - ④ 动作场 **beat-by-beat 拆分**
> - ⑤ 至少 2 个 **narration / caption 框**（慢节奏页 3-5 个）
> - ⑥ 首登角色 **介绍框**（沿用 v1 character_introductions.md）

---

## 1. 总体设计参数（v2 锁定）

| 参数 | v1 → v2 变化 |
|---|---|
| 总页数 | 28 页（不变） |
| 阅读方向 | 日式右→左（不变） |
| 色彩 | 彩色 + 1990 冷色板（不变） |
| 单页面板数 | **v1 平均 3-4 格 → v2 平均 5-6 格**（拆 beats） |
| 每页 caption 数 | **v1 平均 1-2 个 → v2 平均 3-4 个** |
| 时空 caption 频率 | **v1 几乎没有 → v2 每个场景切换必有** |
