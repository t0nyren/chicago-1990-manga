# 《芝加哥1990》人物档案 · 第一阶段（开局·芝加哥南城 → 入行说唱圈 / SBK，约 1~38 章）

> 用途：漫画化人物设定参考。每条按身份/年龄/血统/外貌/服装/性格气质/口头禅与说话方式/关键经历/与主角关系 + 摘原文直接刻画段落（带章节定位）。
> 阶段说明：第一阶段覆盖主角穿越后约两个多月的剧情（家庭生活 → 写《Thrift Shop》卖给小洛瑞 → 入行 → SBK 接盘 → 写《FXXXYOU》给米拉乔沃维奇）。这一段对人物外貌、服装的描写最密集，最适合先做人设稿。

## I. 主角 & 家庭

### 1. 宋亚 / 亚历山大·宋 / "亚力" / "APLUS"
- **身份**：主角，穿越者，原是华国佛系小康青年（码农，刚买房准备结婚），灵魂落入芝加哥南城一名15岁混血男孩身上
- **年龄**：15岁（九年级）
- **血统**：一半华国人血统，一半喀麦隆裔米国黑人，亲生父母死于车祸，寄居在姨妈家
- **外貌**：
  - "略带稚气，长得不错的脸，由于有一半华国人血统，肤色也略淡一些"（第1章）
  - "十五岁身高就有一米七几了，身材也颇匀称"（第1章）
  - 自己剪的清爽利落短发
- **服装演变**：
  - 初期：深色翻毛夹克（内标 U.S.A.F 1969 的空军老兵旧物，磨损厉害）+ 深色中领毛衣 + 干净的浅色牛仔裤和球鞋 + 二手店9.9刀淘到的格纹西装（"喜欢老款式，不爱带厚垫肩的廉价新西装"，第8章）
  - 入行后：棕黄色牛仔皮夹克（前胸+肘部到袖口有骚气流苏）+ 黑色T恤（胸前印日式汉字「勇気」）+ 蓝色牛仔裤 + 经典款匡威帆布鞋（第34章）
- **性格气质**：佛系，谨慎，精于算计但守底线；天启外挂下逐渐转化为"给积分的人"式的精明策划者；嘴上常吐槽，外表装老实
- **口头禅 / 说话方式**：
  - 心中OS极多（"心中暗道""暗暗吐槽"是高频）
  - 关键场合会强行端起架子说"你拦我路？嗯？"配以额头顶人施压（第24章）
  - 兜里揣俩三个25美分镚子防"出紧急状况备着拨打公用电话"（第2章）
- **关键经历**：穿越后十几天 → 数学满分得"APLUS"外号 → 受校园霸凌 → 拍 MV 时在二手店得「天启」灵感 → 写 Thrift Shop → 找古德曼咨询 → 在老乔音乐入行 → SBK 截胡 → 第二次天启写 FXXXYOU → 准备搬出南城
- **金手指设定**：上一世记忆全无；穿越后突击数学超神，但乐感与体育协调性骤降（影响校乐团小号手身份和橄榄球）；偶尔触发"天启"：把未来未问世的经典歌曲完整放入脑海
- **关键经济节点**：穿越第一桶金 4500 USD（卖 Thrift Shop 词的一半 + 封口费 2500+2500）→ 接下来 SBK 给 1 万介绍费 + 10 万对赌保底 + 15 万

**原文摘录（重要刻画）**：
> "按华国人对黑色人种的审美来算的话，他有张略带稚气，长得不错的脸，由于有一半华国人血统，肤色也略淡一些。十五岁身高就有一米七几了，身材也颇匀称……当然，作为穿越者日后发迹是自不待言的"（第1章）

- **口头禅 / 说话方式**：

### 3. 托尼（Tony）
- **身份**：苏茜姨妈的大儿子，主角名义上的表哥兼最常在一起的"哥哥"，校橄榄球队守备组主力

- **服装演变**：
  - 早期：深色大衣（"应是某个工厂的制服，袖口和肘部已快磨没了"，第2章）
  - 跟班期：一身黑西装配墨镜（"昨天我们在老乔音乐对面的地下夜店演了一场……要不是我和'消音器'……"做出明星保镖标志拦截动作，第20章）
- **性格气质**：粗中有细；够义气；脑子时常掉线但关键时刻拎得清；阴谋论患者；嘴上没把门，行动力强；动则掏枪撑场面，但又懂分寸（"在治安好点的篮球场敢亮枪，是因为对面是家境和教育还可以的孩子；换到贫民区他绝对不敢"，第12章）
- **口头禅 / 说话方式**：
  - "FXXX YOU"、"NGer"高频
  - 阴谋论："那帮白人控制了世界……整容你懂吗？脸都可以换……51区……外星人科技……"（第4章）
  - 兄弟话："你的事就是我们的事"（第5章）
- **关键经历**：因体重和身高未达职业线被橄榄球队清退 → 转去给说唱新人小洛瑞当"跟班/伙伴"找出路 → 一身黑西装跟着小洛瑞跑全美宣传
- **关键道具**：腰间别一把黑手枪（第4章篮球场"他掀开T恤下摆，露出别在腰间的黑色枪柄，眼神狠得像迈克泰森"）；'消音器'的丰田86雄鹰是出行工具

**原文摘录（高光镜头）**：
> "托尼像受惊的兔子般一弹而起，却是往场边'消音器'的丰田雄鹰飞奔过去，大概半分钟后又冲回来，一头扎进人堆里，挡在小洛瑞身前……他掀开T恤下摆，露出别在腰间的黑色枪柄，眼神狠得像迈克泰森，'你们想玩多大？NGer!em？你们想玩多大？'"（第4章篮球场对峙）

## II. 老乔音乐 / 小洛瑞团队（南城说唱圈）

### 7. 小洛瑞（Lil Lorri / 洛瑞二世）
- **身份**：芝加哥南城说唱新人，老乔音乐唯一在推的歌手，托尼的儿时好友
- **年龄**：与托尼相仿（约 17 岁），两年前签约后辍学
- **血统**：黑人，但**不是**贫民区出身（住带门禁、物业的高层公寓，铺古典风格地毯）
- **外貌**：
  - "他和托尼年龄相仿，个头比宋亚矮点，非常瘦，相貌算黑人里难得的清秀，头顶脏辫，正戴着耳机一边听歌一边悠闲地拍球热身"（第4章）
  - 不戴任何帮派标志，没有纹身
- **服装**：嘻哈风装扮 + 麦克风；专辑封面上"一手拿着麦克风，一手酷酷的直指前方"（第19章）；MV C 位"老乔的粉西装"对应他的酷帅麦克风
- **性格气质**：略自我中心，傲娇拽拽，被宠坏；现场表演时学乔丹吐舌头庆祝（第4章）；情绪化（输了 battle 在台上哑口、当着众人想哭）
- **口头禅 / 说话方式**：拽，爱答不理；情绪上来"决赛！决赛！我只要决赛！"（第30章）
- **关键经历**：篮球场被人惹 → 托尼拔枪护他 → 写词作曲均自己包办却水平有限 → 二手店保住他的首张专辑 → 抛弃艾尔与老摩根 → 与宋亚关系微妙
- **与主角关系**：表面"兄弟"，实质上是「我是来救你的」与「得了便宜还卖乖」的别扭关系

**原文摘录**：
> "他不像其他说唱歌手，没有纹身，不佩戴任何帮派的标志，歌词里没有什么杀啊毒啊操啊的，连太超过的粗话都没几句"（第6章）
