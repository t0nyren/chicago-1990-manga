# 《芝加哥1990》漫画第一话 · 分镜设计稿

> 用途：第一话的页面级分镜设计 + 节奏安排。读完这份再去看 prompt 文件。

---

## 1. 第一话覆盖范围

**对应原著**：第 1 章「家庭」→ 第 7 章「天启」（约前 200 行）

**为什么这样切**：
- 第 1-7 章构成一个完整的"主角抵达 + 世界建立 + 入行金手指开启"小弧
- 在第 7 章「天启」处自然落幕（最强 cliffhanger：脑海中涌入未来神曲）
- 不切到第 8 章「GOODMAN」是因为律师戏更适合作为第二话的开场（专业转换）
- 涵盖了所有第一话该有的元素：主角魂、家庭、学校、街头、入行触发器、第一个超自然事件

**情绪弧线**：
```
睡梦中被吵醒 → 麻木地适应 → 学校屈辱 → 街头惊吓 → 跟班的困局 → 看似无聊的 MV → 灵感冲入头脑 → 决心
低 ────────── 平 ────────── 低 ────────── 高(惊) ─── 低 ────────── 中 ────────── 高(顿悟) ──── 燃
```

## 2. 总体设计参数

| 参数 | 值 |
|---|---|
| 总页数 | 28 页 |
| 阅读方向 | 日式从右到左（如 Tony 想做条漫/manhwa 风格，告诉我换） |
| 色彩 | **彩色**，沿用已锁定的人设色板：1990 期色调（charcoal / faded indigo / off-white / olive-brown / 浅 caramel 肤色 + 老乔粉作 accent） |
| 风格 | 半写实日漫线 + 节制平涂色块 + 软 cell shading（已锁定） |
| 单页面板数 | 3-5 格为主，关键时刻用整页大格 / 单跨页 splash |
| 文字 | 心声框 / 旁白框 / 对话框 / 拟声 SFX 分类清晰；中文文案（gpt-image-2 文字渲染不稳，最终排版建议人工后期叠 |

## 3. 全 28 页分镜节奏表

### 第一幕：清晨家庭世界（P1–P5，对应原著第 1 章）

**P1 / 整页 splash · 主角醒来**
- 主角侧躺在床上，被楼下骂街声击中头部样
- 楼下传来的话以巨大字体覆盖画面右上角："天知道我多后悔生下了你们！天知道！"
- 旁白框（黑底白字小框）：「这是我穿越后的第十四天。」
- 目标：定调全篇——南城贫民区被吵醒是日常

**P2 / 4 格 · 起床过程**
- 格1：床头近景，主角侧脸睁眼，眉头微皱，无奈
- 格2：他翻身坐起，捞起毛衣往头上套，楼下骂街声继续："M-FXXX 的十七岁！十七岁就 M-FXXX 学会像个碧池一样地夜不归宿了！"
- 格3：心声框："说到碧池，也不知道是谁十七岁的时候连孩子都生俩了！"——同时切到楼下康妮反唇相讥的小插画
- 格4：主角心声 OS："亲生的。"配脸——内心吐槽用的标志性"暗藏小坏笑"

**P3 / 4 格 · 卫生间镜中自己（关键身份介绍页）**
- 格1：走进卫生间，拧开水龙头，水声覆盖楼下骂街
- 格2：镜中半身像——这是漫画里**第一次让读者完整看到主角的脸**，对应人设 sheet 的 face anchor 视角
- 格3：旁白框："按华国人对黑色人种的审美来算的话……" + 主角微微捏自己脸颊的小动作
- 格4：旁白框："穿越者没带前世记忆。除了数学突飞猛进——咱也只能先这么活着。"（淡化处理，为下一格的笑剧做铺垫）

**P4 / 5 格 · 早餐桌（家庭群像首次集合，对应 Multi-Scene Sheet 的 S1）**
- 大格（页面上半）：早餐桌全景，对应已锁定的 S1 场景——苏茜姨妈一手锅铲一手抱弗雷迪、托尼仰头喷奶油、康妮翻白眼倒燕麦、艾米丽倒牛奶、主角入座
- 格2：托尼把奶油罐口转过来："张嘴。"
- 格3：主角："不了。"心声 OS："肥胖绝对是我这一世要时刻警惕的敌人。"
- 格4：托尼："你最近变了很多，亚力。"——主角心一紧
- 格5：康妮插话："多简单的事！为了女人呗！你和谁搞上了？"——成功救场（"再次成功把话题带偏"）

**P5 / 3 格 · 出门**
- 格1：四兄妹推门而出，托尼一把把艾米丽抱怀里给她挡风
- 格2：街景广角——破败的非洲门口小国旗、灰败杂草、生锈栅栏、二月芝加哥的灰蓝天
- 格3：旁白框："这里是芝加哥南城。我的新家。"——节奏短暂下沉

### 第二幕：校车 + 学校屈辱（P6–P10，对应原著第 2-3 章）

**P6 / 4 格 · 校车站台 ET 出场**
- 格1：站台边几个半大小子人手一根烟
- 格2：戴棒球帽的 'ET' 凑近托尼："yooo，托尼，你现在跟小洛瑞混了？"（ET 帽檐下隐约花头巾一角作 careful close-up）
- 格3：托尼笑着把艾米丽送上小学校车，跟 ET 对拳，往一旁走去聊
- 格4：心声 OS："花头巾，不是能随便戴上的东西。"——简单交代帮派世界

**P7 / 4 格 · 校车上 APLUS 命名**
- 格1：校车前排，主角刚坐下，同学："早啊，APLUS！"
- 格2：哄笑特写，几张嘲笑脸
- 格3：主角闭眼装睡，心声 OS："就因为数学考个满分，外号就被改成了'A+'。"
- 格4：心声 OS："华国谁会因为考个满分被孤立？"——配窗外快速倒退的南城街景

**P8 / 4 格 · 教室 + 食堂**
- 格1：课堂，背后小情侣已经抱在一起啃，前面白人男老师念狄金森的诗
- 格2：下课铃，老师夹包闪人
- 格3：食堂，黑人老头看 **Mandela 海报**（1990 年 2 月出狱，27 年牢狱）兴奋大喊："27 年! 迟到的胜利!"——时间线咬合：原稿写 Malcolm X 是笔误（@JoshuaBright 抓出 2026-06-06）
- 格4：主角端餐盘排队，副校长守门口

**P9 / 4 格 · 数学老师办公室**
- 格1：黑人老太数学老师笑眯眯："给个建议——如果你能通过我的九年级基础课程考试，我就把你转到数学高级班里去。"
- 格2：主角："高级班？"
- 格3：旁白框："高级班 = 更多学分。"——配上眼神发亮
- 格4：主角："好的，我会尽力！"

**P10 / 4 格 · 走廊撞见音乐老师**
- 格1：音乐老师（白人中年女性、利落短发）："亚历山大，是吧？上周，你的小号在排练中数次出现明显的错误！"
- 格2：主角内心崩溃，干笑
- 格3：音乐老师："下次合练你别吹小号了，撸铁去吧！"
- 格4：主角小心："撸什么？" / 音乐老师："三角铁！"——cut 到主角拿着小小三角铁的悲剧表情，幽默收尾

### 第三幕：街头篮球与拔枪（P11–P16，对应原著第 4-5 章）

**P11 / 2-3 格 · 上车去打球**
- 格1（大格）：'消音器' 的丰田 86 雄鹰停在校门口，托尼和主角上车
- 格2：广角车内，三人前后排各坐一个，后座球衣护具乱七八糟
- 格3：'消音器' 从驾驶位回头比了个割喉手势——他和托尼刚被橄榄球队清退

**P12 / 3 格 · 篮球场初见小洛瑞**
- 格1：小公园铁丝网篮球场，小洛瑞带着耳机悠闲拍球热身——按人设：脏辫 + 清秀 + 瘦
- 格2：大家碰拳介绍，主角和小洛瑞的复杂碰拳完全失败——主角拳头空对空挥了几下
- 格3：小洛瑞憋着笑拍主角肩："算了，不用在意那些。"——展现两人初见的尴尬

**P13 / 4 格 · 半场对位被虐**
- 格1：主角和小洛瑞单挑，小洛瑞背后运球虚晃转身
- 格2：主角失重一屁股坐地上，小洛瑞已上篮成功，学乔丹吐舌头
- 格3：旁边场地三个黑人小伙走过来："我们来？"
- 格4：小洛瑞答应，主角和托尼下场歇着

**P14 / 大格 + 2 小格 · 拔枪事件**
- 大格（占页面 2/3）：小洛瑞被对方故意沉肘撞倒 → 站起怒推 → 对方推搡 → 冲突扩大 → 这时**托尼半秒钟内冲回'消音器'车里 → 一头扎进人堆**
- 小格1：托尼挡在小洛瑞身前，掀开 T 恤下摆——**黑色枪柄**特写
- 小格2：托尼脸的大特写："**你们想玩多大？NGer！em？你们想玩多大？**"（眼神狠得像迈克泰森）——本话首个高燃镜头

**P15 / 整页大格 · 7 秒寂静**
- 整页一个大格：球场上所有人停在原位，拍球声消失。画面用冷色块覆盖+雪花点效果表达"时间凝固"
- 旁白框："1, 2, 3, 4, 5, 6, 7。"
- 子格只在画面角上点小：被针对的那个人脸上青筋一跳一跳，眼珠子血红

**P16 / 4 格 · 对方撤退**
- 格1：对方两个同伴架着主角推搡那位往后退："算了，算了吧。"
- 格2：小洛瑞从托尼身后闪出，狠拍那人脑袋："娘炮，滚吧！以后别让我再看到你出现在这！"
- 格3：那人额头青筋一跳被同伴拖走
- 格4：主角长舒口气，心声 OS："阿美利加可真是太刺激了。"

### 第四幕：车里谈话 · 跟班经济学（P17–P18，对应原著第 5 章）

**P17 / 4 格 · 驾车归途的责备**
- 格1：丰田车里，主角："你怎么回事！？事情都发生了，你还站在原地看热闹！"——是托尼对主角说
- 格2：主角："为什么？干嘛为那么小的事动枪？冒这么大危险为了什么？值得吗？"
- 格3：消音器从前排蹦句："没有奖学金，没有大学。"
- 格4：托尼："不上大学，我们毕业后能干什么？去 K 记上夜班？还是去商业街扫厕所？"

**P18 / 4 格 · 跟班经济学揭幕**
- 格1：托尼："不行，小洛瑞是我们最好的机会了，这次一定要把握住！"
- 格2：解释跟班传统——画面切到一组小插画蒙太奇：歌手暴富 → 拖着兄弟一起吃香喝辣
- 格3：主角心声 OS："说好听点是兄弟，说难听点不就是马仔、跟班、帮闲么？"
- 格4：主角心声 OS："归根结底，**打工是不可能打工的对吧？**"——本话副标题级金句

### 第五幕：MV 拍摄 · 偶遇产业内幕（P19–P22，对应原著第 6-7 章前半）

**P19 / 2 格 · 十天后周日**
- 格1：标题字"十天后 周日 下午一点"——时间过渡
- 格2：四兄妹出门，托尼搂着身材火辣的前女友前面走，康妮在身后嗤之以鼻

**P20 / 4 格 · 抵达"舞蹈教室"**
- 格1："XX 舞蹈教室"招牌下，'消音器' 一身西装配墨镜守在门口

**P21 / 4 格 · 开拍**

**P22 / 4 格 · 偷听办公室争吵**

### 第六幕：二手店 + 天启（P23–P28，对应原著第 7 章）

**P23 / 3 格 · 杀青后**
- 格1：四点导演喊收工，托尼搂前女友"找个地方"去了
- 格2：AK 偷偷把 $20 塞给主角：「自己人才有。」
- 格3：康妮："我们走，今天老娘要 Shopping！" / 艾米丽："我要买蓬蓬裙！"——主角无奈

**P24 / 4 格 · 走进二手店（对应 Multi-Scene Sheet S2 场景）**
- 格1：二手店外景，"Thrift Store"招牌，铁丝网围着的小店面
- 格2：艾米丽撒腿跑向"蓬蓬裙"区
- 格3：主角穿过一排排衣架，**眼角扫到墙上挂着的粉色西装**——分镜上构图把粉西装放在画面 1/3 位置
- 格4：主角脚步顿了一下："嗯？"——给读者一个"这里有戏"的暗示

**P25 / 4 格 · 三个符号开始打架**
- 格1：主角手伸进口袋，**$20 钞票特写**
- 格2：心声 OS："二十美元，Shopping，粉色西装……"——画面里粉色西装、$20、嘈杂背景音开始有边缘晕染
- 格3：主角眼神空了，背景人物开始虚化
- 格4：主角站定原地——身后康妮还在挑衣服，毫无察觉

**P26 / 整页 splash · 天启降临**
- 整页：主角站在画面中心，**粉西装 + $20 + 麦克风轮廓 + 五线谱波纹** 在他头顶上方汇聚成一个发光符号
- 周围二手店的衣架开始变成原版 MV 镜头的剪影——但都是粗糙速写线，强调"幻觉"
- 没有对白，只有 SFX：低音炮鼓点的拟声（"DOOM DOOM"）+ 萨克斯飘进画面

**P27 / 整页 splash · 神曲涌入脑海**
- 整页：主角双眼睁大占据画面顶部，下半部画面被冲入的歌词雪片状覆盖（中英对照可读）：
  - "I'm gonna pop some tags"
  - "only got twenty dollars in my pocket"
  - "I'm hunting, looking for a come up"
  - "this is FXXXing awesome"
- 心声 OS（贯穿画面中央）："**这是 Thrift Shop。这是——二手店。**"

**P28 / 1 大格 + 1 小格 · 决心 · cliffhanger**
- 大格：主角脸特写，眼睛重新对焦，嘴角隐藏式上翘，手里 $20 攥紧
- 小格：康妮回头问："亚力？发什么呆？"
- 主角："没事。"
- 旁白框（黑底白字大字）：「**——下一话**」
- 副标题位置预留："**第二话 · GOODMAN**"

## 4. 跨页连贯锁

整本第一话以下要素必须**不跨页改变**：
- 主角的脸、发型、五官（按 Identity Sheet v1）
- 主角的衣服：内层深色毛衣 + 浅色牛仔 + 旧白球鞋；P1–P5 室内不穿夹克，P5 出门到 P22 穿 USAF 1969 surplus 夹克，P23–P28 同上
- 配角：托尼的工厂大衣、康妮的鲜艳晚装（拍 MV 当天）、苏茜姨妈的家居袍 + 头巾、艾米丽的睡衣 + 双辫
- 关键符号：粉西装的色温（带一点橙，老乔个人 ID 色）、$20 钞票的方向（必须是同一张）

## 5. 风格统一约束（接下来 prompt 文件会复用）

- 阅读方向：日式右→左
- 画风：半写实日漫线 + 节制平涂 + 软 cell shading
- 色温：冷调底（1990 南城整体偏冷），暖光只在两个位置出现——苏茜家里早晨（P4 的暖黄）+ 老乔粉西装（P24-P28 反复出现）
- 节奏：每页 3-5 格为主，关键 splash 占整页（P1、P15、P26、P27）
- 文字处理：所有对话、心声、旁白框为**汉字**（保留极少英文外号如"APLUS"和歌词原文）。**画面内文字 prompt 里写不写都让 AI 留位置即可，最终建议人工后期叠**
